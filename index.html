☁️ 클라우드 동기화 (Supabase)

다른 기기에서도 같은 데이터를 공유하려면 Supabase 연동을 활성화하세요.
API Key는 Netlify 환경변수에 설정되어 있어 별도 입력이 불필요합니다.

💡 최초 설정 방법 (관리자 1회)
① supabase.com 무료 가입 후 프로젝트 생성
② Project Settings → API에서 Project URL과 anon public key 복사
③ Netlify 사이트 설정 → Environment variables에서 아래 두 변수 등록:
    SUPABASE_URL = https://xxxx.supabase.co
    SUPABASE_ANON_KEY = eyJh...
④ Supabase SQL Editor에서 아래 쿼리 실행:
CREATE TABLE checklist_data (
  id TEXT PRIMARY KEY,
  org_name TEXT,
  admin_pw TEXT,
  teachers JSONB,
  state JSONB,
  updated_at TIMESTAMPTZ DEFAULT now()
);
INSERT INTO checklist_data (id) VALUES ('junior'), ('senior')
ON CONFLICT DO NOTHING;
⑤ 위 체크박스 활성화 후 ✅ 저장
마감된 달은 교사가 수정할 수 없습니다. 관리자는 언제든지 해제 가능합니다.
